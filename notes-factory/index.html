📚 Notes Factory

1 · Build & copy the prompt

Fills your exact TOPIC / EXAM TYPE / SUBJECT into the ExamNotesPDF master prompt (the file itself is never edited) and copies the whole thing — paste it into a new DeepSeek / ChatGPT / Claude chat.

Key terms, definitions, and technical labels always stay in English regardless of this split — this only controls the explanations/tricks/teacher-talk prose.

📱 On phone and the paste comes out cut off? Copy in parts instead

The full prompt is ~20,000+ characters. Some phone keyboards (Gboard/Samsung/SwiftKey clipboard) and mobile browsers silently truncate a single copy that long. Tap "Build & copy prompt" above first, then use these buttons in order — paste each one as its own message in the SAME chat before sending the next. Tell the AI to wait until you say "generation config: go" if it starts replying early.

2 · Paste the AI's reply

Paste the whole reply from DeepSeek/ChatGPT/Claude. The parser reads the machine-readable markers first, and falls back to reading the visible SEO panel + HTML if a model ignored them — nothing is ever a dead end.

3 · Review & edit

Fix anything the parser missed before handing it off. This is what actually gets published.
